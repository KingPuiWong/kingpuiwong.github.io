项目部署

  • 确定项目所需要下载的资源

  • 到各官网上找到你要的资源的下载路径，复制下载连接

  • 到linux服务器的某个路径（个人习惯）,下载到这个目录下

  • 如果需要本地上传资源

  • 如果下载的资源是tar.gz后缀的

  • 如果下载的资源是zip后缀的,现在linux服务器下载unzip命令资源

  • 下载jdk

  • 下载Redis

  • 解压缩好maven后，进行相应的配置

  • 下载mysql的npm包（自己的百度网盘有）

    • 解压后用 yum install -y npm包文件名安装

    • yum list mysql* 找到我们要下载的MySQL服务器

    • yum install -y mysql服务器包名 进行安装

    • systemctl mysqld start 启动MySQL服务

    • systemctl status mysqld 查看MySQL服务器状态

    • grep 'password' /var/log/mysqld.log 查找MySQL服务器刚启动时自动生成的密码

    • mysql -u root -p 使用自动生成的密码进入MySQL服务器

    • alter user root@localhost identified by 'new password'修改MySQL密码

    • 以新密码进入数据库mysql -u root -p

    • 创建你所需的数据库create database ~~~

    • 使用你所创建的数据库,导入sql文件

  • 安装配置Elasticsearch

    • 先解压elasticsearch
    • 再解压elasticsearchIk到elasticsearch的plugins/ik目录(unzip -d)
    • 修改elasticsearch的配置文件（config目录）
